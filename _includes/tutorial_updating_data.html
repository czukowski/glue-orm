Updating data

Say we're not happy with John's second post and we want to replace its content with "-censored-". Assuming we already have a model object available for that post, we would update it like this :

$post->content = '-censored-';
glue::update('post', $post);

...if we had more than one post to update we could also pass an array instead of a single object :

$post1->content = '-censored-';
$post2->content = '-censored-';
glue::update('post', array($post1, $post2));

The above code snippet would cause all fields to be updated in the database. If we only wish to update some fields, they can be provided as a third argument :

glue::update('post', $post, array('content'));
